I. Description
	
	Merci d'avoir téléchargé ConnectedBooks ! ConnectedBooks est une application PHP indépendante permettant de gérer un catalogue important de livres. Sa spécificité est la recherche par ISBN lors de l'import: en entrant le numéro ISBN du livre, l'application se connecte au serveur Européen pour télécharger les informations du livre. Ainsi, la fiche de chaque livre est créée en quelques secondes !
	
	
II. Installation
	
	1. Téléchargez la dernière version de l'application à partir de: http://connectedbooks.web-diot.fr/download.php
	
	2. Dézippez le fichier compressé téléchargé, et placez le dans le dossier connectedbooks de votre serveur.
	
	3. Ouvrez l'assistant d'installation à l'adresse: http://monsite.fr/connectedbooks/install et suivez les étapes.
	
	4. A la fin de l'installation, n'oubliez pas de supprimer le dossier install/ de votre serveur !
	
	
III. Comment utiliser l'appli

	Pour administrer l'application, cliquez sur le lien "Administration" en bas de chaque page. Pour vous connecter, il vous suffit de rentrer les informations de connexion enregistrées lors de l'installation. Vous pouvez alors, à partir de l'administration, gérer tout le site !
